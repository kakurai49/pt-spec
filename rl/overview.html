SIRIUS (Reinforcement Learning)

Overview / 世界観

「Scope → Index → Overview → Spec」の導線で、RL学習を航路化。 Overviewでは、システムの粒度・成果物・テスト観点を地図としてまとめます。

動く仕様書へ Scopeを確認 Homeへ戻る
Scope Index Overview Spec
航路（Scope→Index→Overview→Spec）
Scope Bandit/MDP/Q-learningの到達イメージと、観点束S・A(S)の素案。
Index 14日分の進め方、GitHub運用、PR粒度、seed共有の作法。
Overview 成果物の構造、学習ログの保存先、テスト観点の紐づけ。
Spec ブラウザで動くBanditデモ。累積報酬/後悔・選択回数を即確認。
成果物の構造
レイヤ中身証跡
SpecBanditデモ（HTML/JSのみ）seedとp配列を記録したスクショ/メモ
ModelMDP定義（状態・行動・遷移・報酬）テーブル or markdown図解、Issueリンク
AlgorithmQ-learning更新式とハイパーパラメータテスト観点 S/A(S) と紐づいたノート
OpsGitHub運用（Issue→PR→テスト）PRテンプレ、CIログ（任意）
テスト観点のヒント
  • Bandit：探索率が指定どおり変化するか（seed固定で比較）。
  • MDP：終端状態の扱い（ループ有無）で価値計算が変わることを確認。
  • Q-learning：学習率/割引率/εが0〜1の範囲でクリップされているか。
  • 再現性：seedとp配列がIssue/PRに残り、同じ曲線を描けるか。
ブラウザだけで完結するデモを土台に、バックエンドやPython実装を追加してもOK。まずはここで「動く仕様書」を手に入れてください。
